# BarberShop

Esse projeto visa ajudar uma barbearia fictícia com o cadastro e agendamento de clientes. Os requisitos foram pensados de acordo com o contexto real. 


## Tecnologias Utilizadas

* Java 17 LTS
* NetBeans 18


## Design Patterns  

* MVC Design Pattern
* DAO Pattern
